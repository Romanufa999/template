# Image Generation via Webhook

Генерация изображений для сайта через webhook n8n.

Используй этот скилл когда нужно:
1. Сгенерировать картинку / изображение
2. Создать картинку для сайта, страницы, статьи
3. Отправить задание на генерацию изображения
4. Пользователь говорит "сгенерируй картинку", "нужна картинка", "создай изображение", "сделай фото для сайта", "generate image"

## Endpoint

POST https://gadugestok.beget.app/webhook/generate-insert

## Процедура

1. Сгенерировать случайное имя файла: 10 случайных символов (a-z, 0-9) + `.webp`
2. Заполнить все поля на основе контекста разговора с пользователем
3. Отправить POST-запрос через curl
4. Сообщить пользователю результат и итоговый URL изображения

## Генерация имени файла

```bash
filename=$(cat /dev/urandom | tr -dc 'a-z0-9' | head -c 10).webp
```

## Поля запроса

| Поле | Обязательное | Описание |
|------|-------------|----------|
| filename | да | Случайное имя: 10 символов (a-z0-9) + .webp |
| promt | да | Промпт для генерации изображения (на английском, детальный) |
| title | да | Title картинки для SEO (на языке сайта) |
| alt | да | Alt-текст картинки для SEO (на языке сайта) |
| client | да | Имя клиента / проекта. Спросить у пользователя если не ясно |
| metka1 | нет | Метка/тег 1 (опционально) |
| metka2 | нет | Метка/тег 2 (опционально) |

## Пример запроса

```bash
curl -s -X POST https://gadugestok.beget.app/webhook/generate-insert \
  -H "Content-Type: application/json" \
  -d '{
    "filename": "a7k2m9x4bp.webp",
    "promt": "Modern minimalist kitchen interior with white marble countertops, natural light from large windows, green plants, professional real estate photography style",
    "title": "Современная кухня в минималистичном стиле",
    "alt": "Интерьер современной белой кухни с мраморными столешницами и панорамными окнами",
    "client": "remont-site",
    "metka1": "интерьер",
    "metka2": "кухня"
  }'
```

## Итоговый URL изображения

После отправки, файл будет доступен по адресу:

```
https://s3.ru1.storage.beget.cloud/76ae0220f799-proficient-naida/generate/{filename}
```

## Правила заполнения полей

- **promt**: Всегда на английском. Подробный, описательный. Включать стиль, освещение, композицию, настроение.
- **title**: На языке сайта/проекта. Краткий, SEO-оптимизированный, 5-10 слов.
- **alt**: На языке сайта/проекта. Описательный, для accessibility и SEO, 8-15 слов.
- **client**: Спросить у пользователя если не указан явно в контексте.
- **metka1, metka2**: Категории/теги, если уместно. Можно оставить пустыми.
